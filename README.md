# yespider

------------------------------------

this is easy to get image for spider

------------------------------------

------------------------------------

# install

npm install yespider -g

------------------------------------

------------------------------------

#use

yespider [options]

Options:
  -u, --url      网址                              [default: "www.bilibili.com"]

  -d, --dir      目录                                              [default: ""]

  --help         Show help                                             [boolean]

  -v, --version  Show version number                                   [boolean]
